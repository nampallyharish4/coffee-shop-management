COMMANDS NOW IN README:
-----------------------

Backend with H2 (No MySQL):
```bash
cd backend
mvn spring-boot:run -Dspring-boot.run.profiles=h2
```

Backend with MySQL:
```bash
cd backend
mvn spring-boot:run
```

Frontend:
```bash
cd frontend
npm install
npm start
```

Windows Batch Files:
```cmd
start-backend.bat
start-frontend.bat
RUN_WITHOUT_MAVEN.bat
```
